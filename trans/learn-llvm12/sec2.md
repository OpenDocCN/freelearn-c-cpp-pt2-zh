# 第二部分：从源代码到机器代码生成

在本节中，您将学习如何开发您自己的编译器。 您将从构建前端开始，该前端读取源文件并创建其抽象语法树。 然后，您将学习如何从源文件生成 LLVM IR。 使用 LLVM 的优化功能，您将创建优化的机器码。 您还将了解许多高级主题，包括为面向对象的语言构造生成 LLVM IR，以及如何添加调试元数据。

*   [*第 7 章*](07.html#_idTextAnchor111)，*高级 IR 生成*
